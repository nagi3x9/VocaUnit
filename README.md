# VocaUnit

ボカロP、絵師、動画師、歌い手などが共同で制作・作品投稿を行うイベント「VocaUnit」の公式Webサイト用リポジトリです。

## 隠しコマンド（イースターエッグ）機能について
サイトの画面を開いた状態で、キーボードで特定のボカロキャラクター名（ローマ字）をタイピングすると、サイト全体のテーマカラー（ネオンカラー）およびロゴのグリッチエフェクトがそのキャラクターのモチーフカラーに変化します。

### 対応キャラクターコマンド一覧

| 入力コマンド | キャラクター名 | テーマカラー |
| :--- | :--- | :--- |
| `miku` | Hatsune Miku | ブルーグリーン系 |
| `rin` | Kagamine Rin | イエロー系 |
| `len` | Kagamine Len | イエロー＆ブラック系 |
| `luka` | Megurine Luka | ピンク系 |
| `meiko` | MEIKO | レッド系 |
| `kaito` | KAITO | ブルー系 |
| `gumi` | GUMI | グリーン系 |
| `ia` | IA | ライトパープル系 |
| `una` | Otomachi Una | ブルー＆イエロー系 |
| `kafu` | KAFU | くすみパープル系 |

### リセットコマンド
デフォルトの「シアン＆マゼンタ」カラーに戻すには、昔ながらの以下のコナミコマンドをキーボードで入力してください。

> 上(↑)・上(↑)・下(↓)・下(↓)・左(←)・右(→)・左(←)・右(→)・B・A

※画面右下に現在適用されているテーマ名が表示されます。
